🎪 Como Usar o Sistema

  1. Acesse o DocumentsTab em qualquer lead na pipeline
  2. Arraste arquivos para a área de upload ou
  3. Clique em "Selecionar Arquivos" para abrir o file dialog
  4. Use o botão fallback "Ou clique aqui..." se necessário
  5. Acompanhe o progresso na barra de upload
  6. Gerencie documentos com download e exclusão
